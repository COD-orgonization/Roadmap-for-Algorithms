**Связный список (Linked List)** — линейная структура данных, состоящая из узлов, где каждый узел хранит значение и ссылку на следующий узел. Позволяет эффективно вставлять и удалять элементы без перераспределения памяти.

---

# Объяснение для 5 летнего ребёнка:
Представь, что у тебя есть игрушечный паровозик.

Каждый вагончик — это узелок. Внутри вагончика лежит игрушка (это твои данные).  
У каждого вагончика есть крючочек, который цепляется за следующий вагончик (это ссылка).

Если ты хочешь добавить новый вагончик, ты просто цепляешь его в начало или между другими — не нужно переставлять все остальные.  
Если хочешь убрать один — отцепляешь его, а остальные остаются вместе.

Так работает связный список: много вагончиков-узелков, связанных друг с другом, и ты легко можешь менять их порядок.

# Сравнение временной сложности операций:

|Операция|Связный список|Массив|
|---|---|---|
|Доступ по индексу|O(n)|O(1)|
|Вставка в начало|O(1)|O(n)|
|Вставка в конец|O(n)*|O(1)**|
|Удаление из начала|O(1)|O(n)|
|Удаление из конца|O(n)|O(1)|
|Вставка/удаление в середине (после найденного элемента)|O(1)|O(n)|
*При наличии ссылки на последний узел — O(1).

# Пример кода на Python:

```
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def insert_at_beginning(self, data):
        new_node = Node(data)
        new_node.next = self.head
        self.head = new_node

    def delete_from_beginning(self):
        if self.head is None:
            return
        self.head = self.head.next

    def display(self):
        current = self.head
        while current:
            print(current.data, end=" -> ")
            current = current.next
        print("None")
```

### Пример использования:
```
ll = LinkedList()
ll.insert_at_beginning(3)
ll.insert_at_beginning(2)
ll.insert_at_beginning(1)
ll.display()  # 1 -> 2 -> 3 -> None
ll.delete_from_beginning()
ll.display()  # 2 -> 3 -> None
```
